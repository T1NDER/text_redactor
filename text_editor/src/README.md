# Простой текстовый редактор (Инструкция для чайников)

## Описание
Текстовый редактор на Python с использованием Tkinter.  
Поддерживает базовые операции: создание, открытие, сохранение, поиск, подсчёт слов и строк.

## Требования
- Python 3.7+
- Windows (работает также на Linux/macOS)

## Запуск
1. Перейдите в папку `src`
2. Выполните: `python main.py`

## Горячие клавиши
- Ctrl+N — Новый файл
- Ctrl+O — Открыть
- Ctrl+S — Сохранить
- Ctrl+Shift+S — Сохранить как
- Ctrl+F — Найти
- Ctrl+Z — Отмена
- Ctrl+X/C/V — Вырезать/Копировать/Вставить
